# Data Dictionary

| Term | Definition |
| :--- | :--- |
| **Project/Cycle** | A specific financial period (e.g., Tanam Padi Season 1). |
| **Initial Capital** | Starting fund dedicated to the project. |
| **Transaction** | Financial movement (Income or Expense). |
| **ROI** | Return on Investment: `(Net Profit / Initial Capital) * 100%`. |
| **Remaining Capital** | `Initial Capital - Total Expenses`. |
| **Task** | Action item to complete within a project. |
